    responder.send [channel] [message]

    Sends a message to a certain channel.

    Note

    This does not currently support forwarding commands to other channels. If you'd like that, feel free to come into our support server and ask in the suggestions channel.

    Example

    Input in channel 'test'

    {responder.send;general;lol}

    Output in channel 'general'

    lol
