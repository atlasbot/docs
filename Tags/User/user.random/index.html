    user.random

    Gets the ID of a random user.

    Context Requirements

    • user
    • guild

    Examples

    input

    {user.random}

    output

    111372124383428608

    input

    I pick you, {user.tag;{user.random}}!

    output

    I pick you, Sylver#1058!
