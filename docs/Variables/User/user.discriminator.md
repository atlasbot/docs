# user.discriminator

Returns the users discriminator. The discriminator is that number after your name, for example if your full tag was `Sylver#1058`, your discriminator is `1058`.

## Example

*Input*
```
Hello {user.username}#{user.discriminator}!
```
*Output*
```
Hello Sylver#1058!
```
