# replace (string\> <phrase\> <replacement\)

Replaces occurences of `phrase` in `string` with `replacement`.

## Example

### Input

```json
{replace;Kittens are adorable;Kittens;Puppies}
```

### Output

```json
Puppies are adorable
```
