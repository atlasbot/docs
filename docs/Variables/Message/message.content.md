# message.content

Returns the messages content, not just the arguments. E.g, if the user did `a!yourcommand awd`, it would return `a!yourcommand awd`.

## Example

*Input*
```
Your message content is `{message.content}`
```
*Output*
```
Your message content is `ecksdee`
```
