# guild.createdAt &lt;exact&gt;
		
Returns the date the guild was created. "exact" is a true/false value on whether to include hours/minutes.

### Context Requirements

* guild


### Examples

##### input
```{guild.createdAt}```

##### output
```Jul 31, 2018```


##### input
```{guild.createdAt;true}```

##### output
```Jul 31, 2018, 9:19 PM```
