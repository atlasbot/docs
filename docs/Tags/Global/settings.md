# settings &lt;key&gt;
		
Gets a raw setting value. This is an advanced tag, if you don't understand how to use it or what it does, then you probably shouldn't be using it.

### Context Requirements

* settings


### Examples

##### input
```{settings;prefix}```

##### output
```a!```


##### input
```{settings;plugins.moderation.state}```
