# and *
		
Returns "true" if all args are present, or "false" in any other circumstance.

### Examples

##### input
```{and;; ;test}```

##### output
```false```


##### input
```{or;first;second}```

##### output
```true```


##### input
```{and;false;false}```

##### output
```true```

!!! note
		Booleans are not parsed.
