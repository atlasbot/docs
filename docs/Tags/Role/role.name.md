# role.name [role]
		
Gets the name of a role.

### Context Requirements

* guild


### Examples

##### input
```{role.name;tes}```

##### output
```test```

!!! note
		This assumes the role name is "test" and the input is incomplete.


##### input
```{role.name;340583469149192204}```

##### output
```Developer```

!!! note
		This assumes the ID "340583469149192204" is for the "Developer" role.
