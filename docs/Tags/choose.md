# choose [...options]
		
Returns a random argument.

### Examples

##### input
```{choose;a;b}```

##### output
```a```


##### input
```Cats are {choose;good;bad} mkay```

##### output
```Cats are bad mkay```


##### input
```Doggos are {choose;good;good} mkay```

##### output
```Doggos are good mkay```
