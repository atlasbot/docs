# log moderation [subcommand]

Controls where certain messages are logged in your guild.
			

##### Examples

* a!log moderation [subcommand] #channel
* a!log moderation [subcommand] off


##### Aliases

* a!mod
* a!mods
* a!moderations


##### Plugin
Configuration


##### User Permissions
`Manage Guild`


##### Bot Permissions
