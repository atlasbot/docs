# log log [subcommand]

Controls where certain messages are logged in your guild.
			

##### Examples

* a!log log [subcommand] actions #channel
* a!log log [subcommand] mod #channel
* a!log log [subcommand] mod off


##### Aliases

* a!logs


##### Plugin
Configuration


##### User Permissions
`Manage Guild`


##### Bot Permissions
`Embed Links`, `Manage Webhooks`
