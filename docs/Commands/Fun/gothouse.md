# gothouse &lt;house name&gt;

Gets information about a Game of Thrones house. Run without args to get a random house.
			

##### Examples

* a!gothouse &lt;house name&gt; lannisters
* a!gothouse &lt;house name&gt; starks of winterfell


##### Aliases

* a!goth
