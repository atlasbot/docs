# unban [user] &lt;reason&gt;

Unban a user from the guild.
			

##### Examples

* a!unban [user] &lt;reason&gt; @random because you're being nice now
* a!unban [user] &lt;reason&gt; undefined too cool to ban :^)

Moderation
