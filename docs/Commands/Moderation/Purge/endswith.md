# purge endswith [term] &lt;amount&gt;

Purges messages that end with [term]. Supports wild*cards.
			

##### Examples

* a!purge endswith [term] &lt;amount&gt; xd 10
* a!purge endswith [term] &lt;amount&gt; xd


##### Plugin
Moderation


##### User Permissions
`Manage Messages`


##### Bot Permissions
`Manage Messages`
