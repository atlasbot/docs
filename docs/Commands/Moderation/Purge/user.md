# purge user [user] &lt;amount&gt;

Purges messages that are from the user specified.
			

##### Examples

* a!purge user [user] &lt;amount&gt; @random 10
* a!purge user [user] &lt;amount&gt; @user


##### Aliases

* a!member
* a!from

Moderation
